# Google Forms

## Name

Google Forms

## Category

Assessment and Review

## Location

https://forms.google.com

## Cost

Free

## Description

A form and quiz tool for surveys, exit tickets, assessments, and data collection.

## Pedagogical Uses

Teachers can create quick checks for understanding, reflection prompts, quizzes, or family surveys.

## Teacher Value

Google Forms is flexible, simple, and integrates well with spreadsheets for analysis.

## Artifact

Add exit ticket or quiz example.

- Requires careful settings for quizzes
