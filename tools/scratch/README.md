# Scratch

## Name

Scratch

## Category

Math, STEM, and Coding

## Location

https://scratch.mit.edu

## Cost

Free

## Description

A block-based coding environment for creating animations, games, simulations, and stories.

## Pedagogical Uses

Students can create projects that model math concepts, tell interactive stories, or practice computational thinking.

## Teacher Value

Scratch lowers the barrier to coding while making student thinking visible through creative products.
