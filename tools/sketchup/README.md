# SketchUp

## Name

SketchUp

## Category

Math, STEM, and Coding

## Location

https://www.sketchup.com

## Cost

Free education options with paid plans

## Description

A 3D modeling tool for design, measurement, spatial reasoning, and engineering-style projects.

## Pedagogical Uses

Students can design models connected to geometry, scale, area, volume, and real-world planning.

## Teacher Value

SketchUp connects math to design thinking and visual problem solving.
