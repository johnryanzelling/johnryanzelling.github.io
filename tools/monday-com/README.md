# Monday.com

## Name

Monday.com

## Category

Collaboration and Productivity

## Location

https://monday.com

## Cost

Free trial with paid plans

## Description

A work management platform for boards, timelines, dashboards, automations, and team workflows.

## Pedagogical Uses

Teachers and students can track group projects, plan events, organize tasks, and monitor progress across deadlines.

## Teacher Value

Monday.com gives teams a clear view of responsibilities, status, priorities, and project timelines.

## Artifact

Add project board screenshot or planning workflow.
