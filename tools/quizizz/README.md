# Quizizz

## Name

Quizizz

## Category

Assessment and Review

## Location

https://quizizz.com

## Cost

Free with paid upgrades

## Description

A quiz and lesson platform for live, assigned, and self-paced practice.

## Pedagogical Uses

Students can complete practice quizzes at their own pace while teachers monitor results.

## Teacher Value

Quizizz works well for low-stress review and data-informed reteaching.

## Artifact

Add assigned practice link.
