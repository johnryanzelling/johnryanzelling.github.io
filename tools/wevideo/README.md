# WeVideo

## Name

WeVideo

## Category

Creativity and Media

## Location

https://www.wevideo.com

## Cost

## Description

A cloud video-editing platform for creating videos, podcasts, and multimedia projects.

## Pedagogical Uses

Students can produce video explanations, reflections, tutorials, or project presentations.

## Teacher Value

WeVideo supports media-rich assessment and student voice.
