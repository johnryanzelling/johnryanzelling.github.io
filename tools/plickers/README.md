# Plickers

## Name

Plickers

## Category

Assessment and Review

## Location

https://www.plickers.com

## Cost

Free with paid upgrades

## Description

A formative assessment tool that uses printed cards scanned by the teacher device.

## Pedagogical Uses

Students answer questions without needing individual devices, and teachers collect quick response data.

## Teacher Value

Plickers is useful in limited-device classrooms while still giving immediate assessment feedback.

## Pros

- No student devices needed
- Quick whole-class checks
- Simple routines

## Cons

- Requires printed cards
- Question formats are limited
