# HIGH5TEST

## Name

HIGH5TEST

## Category

Collaboration and Productivity

## Location

https://high5test.com

## Cost

Free with paid upgrades

## Description

A strengths assessment that identifies a user's top five strengths and provides reflection language.

## Pedagogical Uses

Students can reflect on strengths, group roles, collaboration habits, career interests, and personal growth goals.

## Teacher Value

HIGH5TEST can support advisory lessons, team formation, reflection, and strengths-based classroom culture.

## Artifact

Add top five strengths screenshot or reflection activity.
