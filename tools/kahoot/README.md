# Kahoot

## Name

Kahoot

## Category

Assessment and Review

## Location

https://kahoot.com

## Cost

Free with paid upgrades

## Description

A game-based quiz platform for live review and formative assessment.

## Pedagogical Uses

Teachers can run fast-paced checks for understanding, review days, or exit-ticket style games.

## Teacher Value

Kahoot increases participation and quickly shows patterns in student understanding.

Add review game link.

- Some students may feel pressure in timed games
