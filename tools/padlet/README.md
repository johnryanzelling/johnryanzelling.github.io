# Padlet

## Name

Padlet

## Category

Collaboration and Productivity

## Location

https://padlet.com

## Cost

Free with paid upgrades

## Description

A collaborative board for posts, media, links, discussion, and resource collection.

## Pedagogical Uses

Students can share ideas, solve problems, collect resources, or respond to prompts in a shared space.

## Teacher Value

Padlet creates a visible record of class thinking and supports many participation styles.
