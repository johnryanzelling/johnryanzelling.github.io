# GeoGebra

## Name

GeoGebra

## Category

Math, STEM, and Coding

## Location

https://www.geogebra.org

## Cost

Free

## Description

A dynamic mathematics platform for geometry, algebra, graphing, statistics, and 3D exploration.

## Pedagogical Uses

Students can manipulate constructions and observe how mathematical relationships change.

## Teacher Value

GeoGebra supports discovery learning and helps students connect symbolic, visual, and geometric representations.
