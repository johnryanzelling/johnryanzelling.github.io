# ChatGPT

## Name

ChatGPT

## Category

AI and Teacher Productivity

## Location

https://chatgpt.com

## Cost

Free with paid upgrades

## Description

An AI assistant that can help brainstorm, draft, explain, adapt, and refine instructional materials.

## Pedagogical Uses

Teachers can generate lesson ideas, adapt reading levels, create examples, and draft feedback while reviewing outputs carefully.

## Teacher Value

ChatGPT can save planning time and support differentiated instruction when used ethically and transparently.

## Artifact

Add example prompt and revised teacher output.
