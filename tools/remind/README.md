# Remind

## Name

Remind

## Category

Communication and Classroom Management

## Location

https://www.remind.com

## Cost

Free with paid options

## Description

A school communication platform for sending messages to students and families.

## Pedagogical Uses

Teachers can send reminders, announcements, assignment updates, and family communication.

## Teacher Value

Remind helps maintain timely communication while protecting personal contact information.
