# PhET Simulations

## Name

PhET Simulations

## Category

Math, STEM, and Coding

## Location

https://phet.colorado.edu

## Cost

Free

## Description

Interactive math and science simulations from the University of Colorado Boulder.

## Pedagogical Uses

Students can test variables, observe patterns, and connect virtual experiments to academic vocabulary.

## Teacher Value

PhET makes invisible STEM processes easier to explore safely and repeatedly.

Add simulation title and lesson prompt.
