# Wolfram Alpha

## Name

Wolfram Alpha

## Category

Math, STEM, and Coding

## Location

https://www.wolframalpha.com

## Cost

Free with paid upgrades

## Description

A computational knowledge engine that can solve, explain, and visualize math and science problems.

## Pedagogical Uses

Students can check work, compare solution paths, and investigate data or formulas.

## Teacher Value

It can support verification, enrichment, and discussion about why procedures work.

## Artifact

Add sample query and reflection on appropriate use.
