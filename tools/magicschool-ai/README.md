# MagicSchool AI

## Name

MagicSchool AI

## Category

AI and Teacher Productivity

## Location

https://www.magicschool.ai

## Cost

Free with paid upgrades

## Description

An AI platform designed for teacher planning, differentiation, assessment, and communication tasks.

## Pedagogical Uses

Teachers can create rubrics, lesson plans, scaffolds, text-level adaptations, and parent communication drafts.

## Teacher Value

MagicSchool AI is education-specific and can streamline routine planning work.
