# ScreenPal

## Name

ScreenPal

## Category

Creativity and Media

## Location

https://screenpal.com

## Cost

Free with paid upgrades

## Description

A screen recording and video creation tool formerly known as Screencast-O-Matic.

## Pedagogical Uses

Teachers can record mini-lessons, feedback videos, worked examples, or student directions.

## Teacher Value

ScreenPal makes it easier to explain processes asynchronously and personalize feedback.

## Artifact

Add sample walkthrough recording.
