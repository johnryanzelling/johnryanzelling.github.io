# Trello

## Name

Trello

## Category

Collaboration and Productivity

## Location

https://trello.com

## Cost

Free with paid upgrades

## Description

A visual project management board based on lists, cards, labels, and due dates.

## Pedagogical Uses

Students can organize group projects, track tasks, and manage research or design work.

## Teacher Value

Trello supports planning routines and makes group progress easier to monitor.
