# Edpuzzle

## Name

Edpuzzle

## Category

Assessment and Review

## Location

https://edpuzzle.com

## Cost

Free with paid upgrades

## Description

A video lesson platform that embeds questions, notes, and checks for understanding.

## Pedagogical Uses

Teachers can assign instructional videos with pauses for reflection, practice, or accountability.

## Teacher Value

Edpuzzle turns passive video viewing into trackable, interactive learning.

## Artifact

Add video lesson screenshot.
