# Quizlet

## Name

Quizlet

## Category

Assessment and Review

## Location

https://quizlet.com

## Cost

Free with paid upgrades

## Description

A study and flashcard platform with review modes and practice activities.

## Pedagogical Uses

Students can practice vocabulary, formulas, definitions, and key concepts individually or collaboratively.

## Teacher Value

Quizlet gives students flexible retrieval practice and supports quick review routines.
