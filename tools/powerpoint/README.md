# PowerPoint

## Name

PowerPoint

## Category

Creativity and Media

## Location

https://www.microsoft.com/microsoft-365/powerpoint

## Cost

Paid with education access options

## Description

A presentation tool for creating slide decks, visuals, narrated lessons, and student communication products.

## Pedagogical Uses

Students can build presentations, explain processes, organize research, and create visual summaries of learning.

## Teacher Value

PowerPoint supports direct instruction, visual scaffolds, student presentations, and reusable classroom materials.

## Artifact

Add a slide deck example, lesson visual, or narrated presentation.
