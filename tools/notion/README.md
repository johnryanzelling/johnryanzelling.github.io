# Notion

## Name

Notion

## Category

Collaboration and Productivity

## Location

https://www.notion.com

## Cost

Free with paid upgrades

## Description

A flexible workspace for notes, databases, planning pages, resources, and project organization.

## Pedagogical Uses

Teachers can organize course resources, lesson plans, student project hubs, research notes, and collaborative planning spaces.

## Teacher Value

Notion can become a central planning and documentation system for classroom materials and long-term projects.

## Artifact

Add class workspace screenshot or planning template.
