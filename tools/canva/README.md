# Canva

## Name

Canva

## Category

Creativity and Media

## Location

https://www.canva.com

## Cost

Free education options with paid plans

## Description

A graphic design platform for presentations, posters, infographics, videos, and classroom materials.

## Pedagogical Uses

Students can create visual explanations, math posters, infographics, and multimedia projects.

## Teacher Value

Canva supports polished communication and gives students accessible design tools.
