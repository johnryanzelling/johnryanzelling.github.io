# Study.com

## Name

Study.com

## Category

Assessment and Review

## Location

https://study.com

## Cost

Paid with some free resources

## Description

An online learning platform with video lessons, practice questions, study guides, and course resources.

## Pedagogical Uses

Students can use short lessons and practice activities for review, enrichment, remediation, or independent study.

## Teacher Value

Study.com can supplement instruction with structured explanations, practice, and progress-oriented learning materials.
